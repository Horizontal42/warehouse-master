# Warehouse Master

Десктопная утилита для печати нумерованных этикеток в PDF и обновления инвентаря Google Sheets при пересборке ноутбуков.

```
python main.py
```

## Установка

Требования: Python 3.10+, Windows/macOS/Linux.

```
pip install -r requirements.txt
```

## Что умеет

**Вкладка «Labels»** — генерировать PDF с последовательно пронумерованными этикетками в трёх форматах:

| Формат | Размер страницы | Этикеток на страницу |
|--------|-----------------|----------------------|
| Single | 58×40 мм альбом | 1 |
| Grid 2×2 | 58×40 мм | 4 |
| Grid 4×4 | 120×75 мм | 16 |

Задайте один или несколько диапазонов номеров, выберите размер шрифта, нажмите Generate PDF. Включите Auto-print для прямой отправки на принтер. Режим Temporary удаляет файл при закрытии программы.

**Вкладка «Reassembly»** — фиксировать, какие запчасти установлены в ноутбук. По ID ищет каждую запчасть в Google Sheets, отмечает её как отгруженную, записывает грейд в строку ноутбука и пересчитывает категорию ноутбука (NEW/A/B/C/D). Выводит текст для вставки в комментарий и техоперацию.

## Настройка Google Sheets

1. Создайте проект в Google Cloud и включите Sheets API.
2. Создайте OAuth 2.0 Desktop-клиент и скачайте `credentials.json`.
3. Положите `credentials.json` рядом с `main.py` (или в `%APPDATA%/WarehouseMaster/`).
4. Скопируйте `sheet_config.example.json` в `sheet_config.json` и отредактируйте под свою таблицу (имена листов, индексы колонок, значения статусов).
5. Откройте Settings и вставьте URL таблицы.

При первом запуске откроется браузер для подтверждения OAuth; токен сохранится в `%APPDATA%/WarehouseMaster/token.json`.
